Xrely's Autocomplete plugin empowers your site search by provideing aut correction and reach autocomplete feature.

== Description ==

<a href="http://autocomplete.xrely.com">XrelY's Autocomplete</a> is the search solution for web sites.  

<p><strong>Features:-</strong></p>
* Instance on the fly your input correction with respect to your website.
* Empower your site search by search box auto complete.
* Customize search drop down as per your choice.
* Reach autocomplete.

== Installation ==

1. Upload plugin zip content to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to add my own custom css for drop down list? =

Include the css for the drop down list.
e.g.
<code>
        .autocomplete-suggestions{background-color : white;}
    .autocomplete-suggestion:hover{color : red;}
</code>
